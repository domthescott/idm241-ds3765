Triggers

-Hovering over the Home object

-Hovering off the Home object

-Mouse down on the Home object

-Mouse up on the Home object

Rules

Hover on: Cursor turns into a pointer, line extends to the right by 25 viewport width.

Hover off: Pointer turns back into normal cursor, line retracts to the left, disappearing.

Mouse Down: Text and line turn grey and become 5% smaller.

Mouse Up: Text and line return to their original size and color.
